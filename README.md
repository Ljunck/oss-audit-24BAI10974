Description

This project is based on studying Python as a open source software. The aim was to understand how open source works and why it is important in real world. Python was choosen because it is simple, widely used and has a strong community support.



Scripts

 Script 1 – System Report
This script shows basic system information like user, OS, uptime, disk usage and memory details.

 Script 2 – Python Check
This script checks if Python is installed and display its version and location.

 Script 3 – Directory Audit
This script goes through some important system folders and shows their size and permissions.

 Script 4 – Log Analyzer
This script reads a log file, search for a keyword and counts how many times it appears.

 Script 5 – Manifest Generator
This script takes user input and creates a small open source statement saved in a file

Make script executable:
chmod +x script1.sh

Run the script:
./script1.sh
